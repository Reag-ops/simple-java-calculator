# simple-java-calculator
Lightweight Java calculator supporting addition, subtraction, multiplication, division with error handling and modulus.

##Features
**Basic Arithmetic** It performs additionk, substraction, multiplication, division and modulus.
**Divison** It prevents crashes by checking division by zero.
**Interface CLI** It uses java Scanner class to get real-time user input.

##Usage
**The program allows iput of integers as well as the floating point numbers.
**The first line is for inputing the first number.
**The second line is for inputing the operation.
**The third line is for inputing the second number.
